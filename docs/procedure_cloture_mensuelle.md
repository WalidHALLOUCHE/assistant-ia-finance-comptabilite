# Procédure de Clôture Mensuelle

## Objectif
Documenter les étapes et les contrôles à effectuer avant la clôture mensuelle du mois comptable.

## Calendrier
- **Date limite de saisie** : 25 du mois
- **Date de clôture comptable** : 30 du mois
- **Reporting financier** : 5 du mois suivant

## Étapes de la clôture mensuelle

### 1. Vérification des écritures comptables
- [ ] Valider que toutes les écritures du mois sont saisies
- [ ] Vérifier que le total des débits = total des crédits
- [ ] S'assurer que tous les justificatifs sont attachés
- [ ] Contrôler l'absence de doublon

**Responsable** : Chef comptable  
**Délai** : J-5

### 2. Rapprochement bancaire
- [ ] Récupérer les relevés bancaires auprès des banques
- [ ] Pointer les écritures comptables avec les mouvements bancaires
- [ ] Identifier les écarts non rapprochés
- [ ] Générer les écritures de régularisation

**Responsable** : Trésorier  
**Délai** : J-3

### 3. Contrôle des factures fournisseurs
- [ ] S'assurer que toutes les factures du mois sont comptabilisées
- [ ] Vérifier l'absence de factures doubles
- [ ] Identifier les factures déjà payées
- [ ] Lister les factures à payer

**Responsable** : Comptable Fournisseurs  
**Délai** : J-4

### 4. Validation des comptes clients
- [ ] Pointer les factures avec les paiements reçus
- [ ] Identifier les factures impayées
- [ ] Mettre à jour la provision pour créances douteuses si nécessaire

**Responsable** : Comptable Clients  
**Délai** : J-4

### 5. Analyse budget vs réel
- [ ] Comparer les charges réelles avec le budget
- [ ] Identifier les écarts > 10%
- [ ] Justifier les écarts significatifs

**Responsable** : Contrôleur de Gestion  
**Délai** : J-2

### 6. Clôture comptable
- [ ] Générer la balance avant clôture
- [ ] Vérifier que débit = crédit
- [ ] Enregistrer les écritures de régularisation
- [ ] Générer la balance de clôture

**Responsable** : Chef comptable  
**Délai** : Jour 1 du mois suivant

### 7. Reporting et commentaires de gestion
- [ ] Générer le reporting financier
- [ ] Rédiger les commentaires de gestion
- [ ] Présenter au management

**Responsable** : Directeur Financier  
**Délai** : J+5

## Documents clés
- Balance avant clôture
- Balance de clôture
- Rapprochement bancaire
- Listing des factures non payées
- Analyse budget vs réel
- Rapport de clôture

## Contrôles de qualité
- Aucune écriture en brouillon ne doit subsister
- Tous les justificatifs doivent être archivés
- Tous les écarts doivent être expliqués
